Seeta Alumni Merchandise Upload Assets
====================================

This folder is organized for uploading merchandise mockups into the current Seeta Alumni system.
Each merchandise category has its own folder. Each product color/variant has its own separate PNG image.

Folders:
- 01_caps
- 02_hoodies
- 03_tshirts_unisex
- 04_tshirts_women
- 05_polo_shirts
- 06_wristbands
- 07_tote_bags
- 08_stickers
- 09_collection_overview

Notes:
- Caps were corrected to remove side logos/side patches.
- Polo shirt mockups follow the real SHOSA wording style: SHOSA + SEETA HIGH OLD STUDENTS ASSOCIATION.
- Wristband mockups follow the real wristband wording style from the sample photos.
- _full_*_sheet.png files are optional full catalogue/banner images; use the named variant PNG files for individual products/options.
- See manifest.csv for every file and recommended use.
