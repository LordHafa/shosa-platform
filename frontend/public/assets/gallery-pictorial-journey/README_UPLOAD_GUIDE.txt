SHOSA Pictorial Journey Images - Optimized System Upload Folder

This folder contains images extracted from the SHOSA Pictorial Journey PDF, organized into separate event/theme folders under one universal folder.

Use these folders to append images into the Seeta Alumni system Gallery/Admin Gallery.

Recommended upload approach:
1. Open the relevant event/theme folder.
2. Upload images from that folder under a matching category/title in the admin gallery.
3. Use manifest.csv to trace each image to its original PDF page.

Notes:
- Images are optimized JPEGs for easier upload and smaller file size.
- The original large PNG extraction folder was reduced for browser/system upload use.
- The folder structure preserves the event categories from the SHOSA journey PDF.
