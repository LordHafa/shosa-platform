# SHOSA Platform User Manual

## Public visitor
1. Open the homepage.
2. Browse events, gallery, store, about and contact pages.
3. Register as alumni to access member features.

## Alumni registration
1. Go to Register.
2. Fill first name, last name, optional other names, email, phone and school details.
3. Upload a profile photo if available.
4. Submit and login.

## Alumni dashboard
The dashboard shows your profile photo, profile completion, SACCO status, payment activity, store shortcut, events and gallery links.

## SACCO application
1. Login as alumni.
2. Open Join / Update SACCO.
3. Review the locked Regular membership status.
4. Choose a savings plan: monthly, quarterly or decide later.
5. Save application.
6. Proceed to pay the required UGX 50,000 membership registration fee.

## SACCO payments
1. Open Make Payment.
2. Choose payment type.
3. Amount is locked for fixed SACCO obligations.
4. Choose MTN or Airtel where enabled.
5. Enter phone and transaction reference/proof where required.
6. Submit payment request/proof.

## Admin
Admins use the unified login flow. Public pages do not expose an admin link.
