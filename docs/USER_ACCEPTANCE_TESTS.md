# SHOSA User Acceptance Test (UAT) Checklist

## Public pages
- [ ] Homepage loads.
- [ ] Theme toggle appears bottom-right and changes homepage.
- [ ] Store, events, gallery, about, contact pages load.
- [ ] Footer has no public Admin link.

## Authentication
- [ ] Alumni registration works.
- [ ] Login works.
- [ ] Navbar hides Login/Register after login.
- [ ] Logout redirects to alumni login.
- [ ] Protected alumni pages redirect when logged out.

## SACCO registration
- [ ] Membership type is locked to Regular for public applicants.
- [ ] Founder/Executive cannot be selected publicly.
- [ ] Savings plan dropdown updates amount and system purpose automatically.
- [ ] Saving application redirects to first-fee payment when needed.

## SACCO payments
- [ ] First payment locked to UGX 50,000 membership fee.
- [ ] Monthly savings locked to UGX 50,000.
- [ ] Quarterly savings locked to UGX 200,000.
- [ ] Extra savings/top-up allows custom amount.
- [ ] MTN button says Pay with MTN Mobile Money.
- [ ] Airtel button says Pay with Airtel Money.
- [ ] Payment proof/reference fields are readable.

## Admin
- [ ] Admin dashboard loads.
- [ ] SACCO analytics cards/charts load.
- [ ] Alumni/SACCO/Payments tabs load.
- [ ] Gallery admin page loads.
