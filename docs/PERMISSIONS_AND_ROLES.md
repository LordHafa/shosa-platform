# SHOSA Permissions and Roles

## Public visitor
- View public pages.
- Register/login.
- Browse store/gallery/events.

## Alumni
- Access dashboard/profile.
- Apply for SACCO.
- Submit payment requests/proofs.
- View own payment history.

## SACCO member
- Alumni with SACCO application and confirmed first membership fee.
- Make savings/subscription payments.
- View SACCO dashboard.

## Admin
- View alumni, SACCO members and payments.
- View SACCO analytics.
- Manage gallery.

## SACCO staff / Treasurer (recommended)
- Verify payment proofs.
- Reject suspicious submissions.
- Export payment/audit reports.

## Auditor (recommended)
- Read-only access to payment records, audit trail and SACCO reports.

## Super Admin
- Assign admin roles.
- Assign elevated SACCO categories such as Founder/Executive.
- Manage system settings.

## Security rule
Admin and privileged features must not be linked from public footer/nav.
