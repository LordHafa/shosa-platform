# SHOSA UI Brand System 2026

This release refreshes SHOSA into a youthful, vibrant, Gen-Z friendly, modern corporate platform while preserving the core SHOSA blue and gold identity.

## Principles

1. **Blue remains the trust color**: Used for institutional identity, navigation, hero backgrounds, and primary surfaces.
2. **Gold remains the prestige color**: Used for calls-to-action, highlights, SACCO trust signals, and premium accents.
3. **Glass panels unify the site**: Cards, dashboards, tables, and forms use soft translucent surfaces to feel modern without losing readability.
4. **Theme toggle must always be readable**: Classic mode is deep and premium; Bright mode is energetic and youthful.
5. **Financial pages stay audit-first**: SACCO workflows remain structured, controlled, and admin/gateway verified.
6. **Admin is an operations center**: The dashboard now has command-center cards for verification, broadcasts, security, and audit posture.

## Theme Modes

- **Classic**: Dark, premium, executive, high contrast.
- **Bright**: Blue-forward, youthful, campus-energy, still corporate.

## Updated Areas

- Homepage theme compatibility and visual refresh
- Public pages global visual consistency
- SACCO pages readability and controls
- Admin dashboard operations-center upgrade
- Form fields and buttons readability
- File picker styling consistency
- Mobile responsive alignment

## Testing Expectations

Every page should be tested in both Classic and Bright modes for:

- readable buttons
- readable cards
- readable form fields
- readable tables
- visible focus states
- no hidden footer/admin links
- no unusable SACCO controls
